# mahakumbh 2025
Spiritual Insights: Mahakumbh 2025-26 Dashboard

# 2. Short Description and Purpose
This Power BI dashboard provides analytical insights into attendee participation, accommodation preferences, transportation modes, demographic distribution, and visitor feedback during Mahakumbh 2025-26.
The project was developed to demonstrate data visualization, dashboard design, data cleaning, and business intelligence techniques using Power BI.
Note: The actual Mahakumbh 2025 witnessed approximately 67 crore visitors. Since real-world attendee-level data is not publicly available, a synthetic dataset was generated using AI tools (Gemini AI) for educational and analytical purposes. The original dataset contained approximately 200,000 records, which were cleaned, transformed, and optimized to a final dataset of 40,000 records for efficient analysis and visualization.
